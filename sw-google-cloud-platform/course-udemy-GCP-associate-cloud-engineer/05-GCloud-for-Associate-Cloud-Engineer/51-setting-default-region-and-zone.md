## Compute Instances - Default Region and Zone

> - Three options:

```sh
Option 1
    - Centralized configuration.
        gcloud compute project-info add-metadata

Option 2
    - Local gcloud configuration.
        gcloud config set compute/region REGION

Option 3
    - Command specific.
        --zone or --region
```
